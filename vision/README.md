# Vison Examples

Targeting vision applications.

## Pre-requisite 

1. Install OpenCV 4.4.0 (Optional - Needed for Vitis Vision Libraries): Check instruction [here](Install_vitis_vision_lib.md)

2. Update Submodules (Optional - Needed if using Vitis Vision Library examples) using,

        git submodule update --recursive --init

    This will clone Vitis_Libraries if not cloned.
